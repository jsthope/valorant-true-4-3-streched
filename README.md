# Config
You need to edit these 3 fields:
```py
WIDTH = 1440
HEIGHT = 1080
HZ = 165
```
# Usage
1. Put the game in full windowed screen of your native screen resolution (probably 16:9).
2. Restart the game.
3. Run ```python3 res.py``` in your cmd
